Convert map coordinates to Geocentric (XYZ) coordinates.

Usage

globe(x, ...)

# S3 method for default
globe(x, gproj = "+proj=geocent +ellps=WGS84", ...)

Arguments

x
list of tibbles, in rangl form
...
arguments to methods (none used)
gproj
Geocentric PROJ.4 string, defaults to WGS84

Value

rangl object with vertices table modified

Description

Convert map coordinates to Geocentric (XYZ) coordinates.

Examples

library(maptools)

Checking rgeos availability: TRUE

data(wrld_simpl)
g <- globe(rangl(as(wrld_simpl, "SpatialLinesDataFrame")))
plot(g, lwd = 3)
