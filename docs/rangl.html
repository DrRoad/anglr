Usage

rangl(x, ...)

# S3 method for SpatialLines
rangl(x, ...)

# S3 method for SpatialMultiPoints

# S3 method for SpatialPoints

# S3 method for trip
rangl(x, ...)

# S3 method for mesh3d
rangl(x, ...)

Arguments

x
input data
...
arguments passed to methods
max_area
maximum area in coordinate system of x, passed to triangulate 'a' argument

Value

a list of tibble data frames, using the gris-map_table model

Description

Create primitive-based "mesh" structures from various inputs.

The 'rangl' package helps transcend the following problems in R and GIS generally:

Details

#' Methods exist for SpatialPolygons, SpatialLines, rgl mesh3d(triangle) ...

  • coordinates beyond X and Y, or longitude and latitude
  • attributes stored where appropriate, on vertices, primitives, branches, or objects
  • topology and geometry are not conflated
  • spatial data can be properly represented as a graph of spatial primitives
  • polygons as true surfaces, not just glorified lines with a path-filling rule
  • TBD higher dimensional primitives are possible
  • TBD n-dimensional rasters with curvilinear coordinates, and the discrete-continuous distinction

I. Creation

rangl
create a rangl table set from various input types

II. Plotting

globe
convert X,Y planar or angular to 3D on the surface of a globe, based on the data in longitude-latitude form
plot.trimesh
plot 2D topology in 3D geometry space
plot.linemesh
plot 1D topology in 3D geometry space
plot.pointmesh
plot 0D topology in 3D geometry space
